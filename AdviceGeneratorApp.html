Visão Geral do Projeto

Este projeto foi realizado como parte de um desafio do Frontend Mentor, focado em reforçar meus conhecimentos em consumo de APIs e criação de interfaces interativas. A proposta era simples: buscar conselhos aleatórios de uma API pública e exibi-los de forma dinâmica. Para isso, desenvolvi toda a estrutura do projeto, estilização e lógica de funcionamento, garantindo uma experiência fluida e responsiva para o usuário.

Principais funcionalidades implementadas

  • Integração com a Advice Slip API para obter conselhos aleatórios;
  • Atualização do conselho exibido de forma dinâmica, sem recarregar a página;
  • Botão para gerar novos conselhos a cada clique;
  • Manipulação eficiente do DOM para exibir as informações;
  • Layout totalmente responsivo, adaptado a diferentes dispositivos;
